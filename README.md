## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

Installed Node.js

### Installation

Clone the repo
```sh
git clone https://github.com/krzysztofrusiecki/trivia-challenge.git
```
Inside project folder install NPM packages
```sh
npm install
```

### Run application

Run the app
```sh
npm start
```
